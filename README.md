wtanaka.inkscape
================

Ansible role to install Inkscape.

Inkscape is a professional vector graphics editor for Windows, Mac OS
X and Linux. It's free and open source.


Example Playbook
----------------

    - hosts: all
      roles:
         - wtanaka.inkscape
